AI HNSW OPTICS:

In this project I implemented a HNSW powered OPTICS clustering algorithm.

The HNSW algorithm is used to approximate the nearest neighbors problem in a short runtime.
OPTICS is an AI unsupervised learning clustering algorithm. It is an extension of the DBSCAN algorithm.

The algorithm is very easy to use. A user just has to input a dataset, epsilon distance and minPts. These are the same parameter as for the standart OPTICS and DBSCAN algorithm. In the first step, the user can see the dataset in a 2D plot. Then the algorithm creates a reachability plot. The user can based on the reachability set a limit reachability for the groups forming clusters. Based on the limit reachability the clusters than are formed.

With its runtime, the algorithm is a state-of-the-art clustering algorithm. As you can see in the example runs, the approximated clusters are often very precise.

The HNSW OPTICS is a: density based approximated nearest neighbors clustering algorithm.
