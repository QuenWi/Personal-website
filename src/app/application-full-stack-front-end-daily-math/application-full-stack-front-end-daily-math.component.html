Daily Math:

For some daily math questions, I programmed an android app using android studio. The purpose of the app is to learn how to do simple math questions in your head.

The application's interface is thoughtfully designed to ensure ease of use, making it a perfect fit for users of all ages. Its simplicity is a key feature, allowing you to dive into your daily math challenge without any problems.

The user can choose between 3 difficulties. Then the user has to answer 10 math equations. Afterwards, the results are shown.
