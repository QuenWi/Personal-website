ToDo-List:

To keep your head empty, it can help to export thoughts/To-Does to a list. That's why I created a to-do list, which I start by booting Windows.

My goals were to create an easy-to-use and fast to-do list. For that, I only added minimalistic functionality The program has everything you need and not more. Therefore, everyone, no matter their situation, can use it without any introduction.

I personally recommend adding this program as a startup application. This way, you can make the most of its features.

The user can add ToDos with or without priority, move ToDos around and remove them.

Code available on: Github Link

The program is a .Net Visual Studio project.
