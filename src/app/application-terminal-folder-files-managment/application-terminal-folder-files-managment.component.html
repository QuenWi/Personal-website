Folder-Files managment:

In every operating system, you only have a certain way to rename a bunch of files I wanted to rename every file with a certain given value like lastWriteTime/lastAccessTime/creationTime. By renaming the files in this certain way, they are displayed in every operating system in the same order.

This program is renaming in every subfolder all the files after the folder. Furthermore, this program includes option for a blacklist and an option to link all new files.

Also, this program has some protection to not accidentally delete files. If a file can't be renamed, then this program runs in troubleshooting mode.

As a protection to only allow renaming files in certain folders, the user has to set up a folder first. If the program runs for the first time on a new folder, then it asks the user if he/she wants to set up the folder. Then two additional folders for new files (000NewFiles) and a folder for logs (000FolderFilesManagement) are created.
