Spotify playlist converter:

Whenever you have a public playlist, you can use the program to save the titles of the playlist in a .csv file.

Furthermore, you can save private playlists by making them for a few minutes public.

The application gets all titles from a public playlist and saves all the song names and their artist in a file.

This is perfect for creating a backup of public playlists. Furthermore, you can create other programs that use this file.

The user only has to input the link to a public playlist and a folder to save the .csv file.

Code available on: Github Link

The program is a .Net Visual Studio project. You only have to add in the script you spotify developer key.
