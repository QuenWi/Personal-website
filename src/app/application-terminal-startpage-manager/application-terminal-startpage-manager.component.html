Startpage manager:

Each of us uses certain websites very often. For that reason, I created a program that allows me to create and edit my own personalized start page.

In the program you can add headlines and links to your start page. Both of them you can give an daily time restriction.

The user can manage the start page with an easy terminal application, allowing you to effortlessly add, modify, or remove links as your interests evolve.

I recommend making the created webpage your start page. It is very useful for daily usage.

Code available on: Github Link

The program is a .Net Visual Studio project.
