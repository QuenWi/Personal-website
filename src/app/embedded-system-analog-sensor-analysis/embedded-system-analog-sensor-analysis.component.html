Embedded Systems Analog Sensor Analysis:

In this project I create a scheduler for the ATmega328. The scheduler uses a priority queue to manage tasks.

I'm using the ATmega328P and the AVR-GCC cross compiler. For visualisation of the data, I used pyhton.

Connected to the ATmega328P is a photoresistor. In this programm, I send multiple ADC values via UART to my computer. My computer than creates a graph, which plots the ADC values. By this you can test the sensor and use the values to create in further projects.

At Image 3 we can see the influence of a smartphone light on the photoresistor and what happens when my hand covers the registor. This can now be used to implement functionality, based on the sensor data.
